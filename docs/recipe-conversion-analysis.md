## Conversion Script Approach

For auto-convertible recipes, a script can:

1. Extract intro (content before `## ingrediënten`)
2. Extract ingredients (list after `## ingrediënten` until next `##`)
3. Extract instructions (paragraphs after `## bereiding`)
4. Extract outro (optional content after main instructions)
5. Generate structured front matter
6. Replace file content with just front matter

For recipes with extra sections, manual review is needed to decide if they should be:
- Merged into main instructions
- Added as `sections` in front matter
- Kept in markdown body
