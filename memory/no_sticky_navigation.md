User does not want sticky navigation (header that stays visible while scrolling).

**Why:** User explicitly rejected this change during a website analysis session, stating "I don't like it."

**How to apply:** Do not propose sticky navigation as a UX enhancement. If analyzing website performance or UX, skip this optimization. Other navigation improvements (scroll-to-top button, breadcrumbs) are acceptable alternatives.
